Java IO Streams

26 October 2015

Java Input Output streams abstract different ways in which you can input and output data. The stream can be sent to or taken from any source, it does not matter. It is really a stream or flow of bytes from one destination to another.

Java divides streams into bytes or character based categories. The character unit of a character based stream is a Unicode or ASCII character and transfers input and output of 8-bit bytes at a time.

This is in contrast to FileReader and FileWriter, which even though they internally use FileInputStream and FileOutputStream, read and write two bytes at a time. This can slow down your program and so it can be an advantage to use FileInputStream and FileOutputStream directly to transfer the data across sources.
